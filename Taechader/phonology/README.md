|-|:-:|:-:|:-:|:-:|:-:|:-:|:-:|
|비음|m||n|ɲ||ɴ||
|무성 파열음|p||t||c|q|(ʔ)|
|유성 파열음|b||d||ɟ|ɢ||
|무성 파찰음|||ts|tɕ||||
|유성 파찰음|||dz|dʑ||||
|무성 마찰음|f||s|ɕ|ç|χ|h|
|유성 마찰음|v|ð||||ʁ||
